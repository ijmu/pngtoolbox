    Choose the right operation

    Edge removal or global color removal?

    These modes solve different background problems. Choosing the safer mode first protects colors inside the subject.

    ModeUse it whenTradeoff
    Connected to edgesThe subject is surrounded by one background colorSafer for white details enclosed inside the subject

    FAQ

    WebP to PNG questions

    Are WebP files uploaded?

    No. For normal use, selected WebP files are converted locally in your browser.

    Does PNG keep transparency?

    Yes. If the WebP image contains transparent pixels, the PNG output should keep that transparency in browsers that support WebP decoding.

    Why is PNG larger than WebP?

    PNG is designed for lossless graphics and transparency, while WebP often uses stronger compression for web delivery.

    Can animated WebP files be converted?

    This browser tool is intended for still images. Animated WebP files may only expose a single frame depending on browser behavior.
